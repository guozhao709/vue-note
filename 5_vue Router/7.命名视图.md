# 命名视图

比如, 我们在一些案例中, 会直接在 app.vue 中的 template 中直接写 router-view, 这时只能有一个 main 视图, 若我们需要三栏布局, 则:

```html
<router-view class="view left" name="left"></router-view>

<!-- 默认为default -->
<router-view class="view left"></router-view>

<router-view class="view right" name="right"></router-view>
```

在 router 中

```js
{
    path: '/',
    // 复数形式
    components: {
        default: Main,
        left: LeftSidebar,
        right: RightSidebar
    }
}
```

## 嵌套命名视图

```
/settings/emails                                       /settings/profile
+-----------------------------------+                  +------------------------------+
| UserSettings                      |                  | UserSettings                 |
| +-----+-------------------------+ |                  | +-----+--------------------+ |
| | Nav | UserEmailsSubscriptions | |  +------------>  | | Nav | UserProfile        | |
| |     +-------------------------+ |                  | |     +--------------------+ |
| |     |                         | |                  | |     | UserProfilePreview | |
| +-----+-------------------------+ |                  | +-----+--------------------+ |
+-----------------------------------+                  +------------------------------+
```

在 template 中:

```html
<h1>User Settings</h1>
<nav-bar />

<router-view></router-view>
<router-view name="down"></router-view>
```

在 router 中:

```js

{
    path: '/settings',
    component: UserSettings,
    children: [
        {
            path: 'emails', // 会渲染到第一router-view中
            component: UserEmailsSubscriptions
        },
        {
            path: 'profile',
            components:{
                default: UserProfile,
                down: UserProfilePreview
            }
        }
    ]
}

```
