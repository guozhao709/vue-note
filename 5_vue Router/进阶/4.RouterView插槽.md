# RouterView 插槽

```html
<router-view v-slot="{ Component }">
  <component :is="Component"></component>
</router-view>
```

等同于

```html
<router-view></router-view>
```

## keepAlive

```html
<router-view v-slot="{ Component }">
  <keep-alive>
    <component :is="Component"></component>
  </keep-alive>
</router-view>
```

## transition

```html
<router-view v-slot="{ Component }">
  <transition>
    <component :is="Component"></component>
  </transition>
</router-view>
```

也可以在 transition 中嵌套 keep-alive

也可以传递 props 和插槽, 但实际开发中不会这样做, 会导致同等级路由都使用相同的 props 和插槽

## 模板引用

```html
<router-view v-slot="{ Component }">
  <component :is="Component" ref="myComponent"></component>
</router-view>
```
