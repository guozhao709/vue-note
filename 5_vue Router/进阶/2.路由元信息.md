# 路由元信息

是定义在路由中的的一个属性: meta, 一般用来储存与路由相关的信息

```javascript

{
    path: '/foo',
    component: Foo,
    meta: {title: 'Foo Page', requiresAuth: true},
    children: [
        {
            path: 'bar',
            component: Bar,
            meta: {title: 'Bar Page', requiresAuth: false}
        }
    ]
}
```

如何访问meta:

```javascript
import { useRoute } from 'vue-router';


const route = useRoute();
const meta = route.meta;

// 或者
const meta1 = route.matched[0].meta;
```
