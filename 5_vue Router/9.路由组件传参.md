# 路由组件传参

## 将 props 传递给路由组件

我们要在组件上访问参数, 之前用

```template
<h1>UserID: {{ $route.params.id }}</h1>
```

```js
{
    path: '/user/:id',
    component: User,
}
```

可以使用 props 来传递参数:

```vue
<script setup>
defineProps(["id"]);
</script>
<template>
  <h1>UserID: {{ id }}</h1>
</template>
```

```js
{
    path: '/user/:id',
    component: User,
    props: true,
}
```

## 布尔模式

当 props 设置为 true 时，route.params 将被设置为组件的 props。

## 对象模式

```js
const routes = [
  {
    path: "/promotion/from-newsletter",
    component: Promotion,
    props: { newsletterPopup: false }, // <--- 静态 props 对象
  },
];
```

在 vue 中:

```js
defineProps({
  newsletterPopup: {
    type: Boolean,
    default: true,
  },
});
```

如果这个组件作为路由组件, 那么 newsletterPopup 将会为 false

## 函数模式

```js
{
    path: '/'
    component: Home,
    props: (route) => ({ query: route.query.q })
}
```

此时, url: `/?q=vue` 会将 query 参数传递给 Home 组件的 props.query 属性.

## 通过插槽(不建议)

```template
<RouterView v-slot="{ Component }">
  <component
    :is="Component"
    view-prop="value"
   />
</RouterView>
```
