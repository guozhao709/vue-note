# 重定向和别名

## 重定向

重定向 redirect 有三种写法:

1. 字符串形式

```js
{
    path: '/home',
    redirect: '/'
}
```

2. 对象形式

```js
{
    path: '/home',
    redirect: { name: 'home' }
}
```

3. 函数形式

```js
{
    path: '/home',
    redirect: ()=>{
        return '/login' // 这里可以返回一个 path 字符串或者一个路由对象
    }
}
```

注: 除非有 children 属性, 否则重定向一般不需要 component 属性。

### 相对重定向

```js
{
    path: '/home/:id/posts',
    redirect: ()=>{
        return to.path.replace(/posts$/, 'profile');
        // 正则, $ 匹配字符串末尾, 所以替换 posts 结尾的路径为 profile
    }
}
```

## 别名

使用重定向时, url 为/home, 但在浏览器地址栏显示的是/, 为了解决这个问题, 可以使用别名 alias:

```js
{
    path: '/',
    component: Home,
    alias: '/home'
}
```

多个别名

```js
{
    path: '/user',
    component: User,
    children: [
        // url: /user 或 /people 或 /user/list
        {
            path: '',
            component: UserList,
            alias: ['/people', 'list']
        },
    ]
}

```

有参数时, 绝对路径的别名要包括参数
