# 命名路由

路由对象中可以有 name 属性

```javascript
{
    path: "/main",
    name: "main",
    component: Main
}
```

注: name 是唯一的

好处:

```javascript
router.push({ name: "main" });
router.push({ path: "/main" });
```

```html
<router-link :to="{ name: 'main' }">Go to Main</router-link>
```

在有 parmas 时比用 path 时方便
