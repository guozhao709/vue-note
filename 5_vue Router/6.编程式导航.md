# 编程式导航

## router.push()

使用 router.push()方法可以实现编程式导航。

```javascript
router.push({ name: "user", params: { id: 123 }, query: { sort: "desc" } });
```

当然, 可以使用 path, 但不建议

```javascript
router.push("/user/123?sort=desc");
//或者
router.push({ path: "/user/123", query: { sort: "desc" } }); // parmas与path不能同时使用
```

当指定 parmas, 可以指定字符串、数字、和数组（重复参数），其他类型会转化为字符串。

## router.replace()

router.replace()方法可以实现替换当前路由，不会留下历史记录。

或者 router.push({replace: true})

## 横跨历史

router.go()方法可以实现横跨历史记录。

```javascript
router.go(-1); // 后退
router.go(1); // 前进
```
